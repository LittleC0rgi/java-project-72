# Анализатор страниц

Учебный проект на Java.

## Быстрый старт

Клонируйте репозиторий и выполните:

```bash
# Собрать проект
make build

# Запустить приложение
make run
```

Или собрать дистрибутив и сразу запустить:

```bash
make build-and-run
```

## Доступные команды

```bash
make build     # собрать проект
make run       # запустить приложение
make test      # запустить тесты
make lint      # проверить стиль кода
make report    # сформировать отчёт покрытия тестами
make clean     # очистить проект
make install   # собрать готовый дистрибутив
make run-dist  # запустить собранный дистрибутив
```
