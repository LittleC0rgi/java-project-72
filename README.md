# Анализатор страниц

Учебный проект на Java

## Быстрый старт

Клонируйте репозиторий и выполните команды:

```bash
# Собрать проект
make build

# Запустить проект
make run-dist
```

Или собрать и сразу запустить одной командой

```bash
make build-and-run
```

> Требуется OpenJDK версии 21 и Gradle версии не ниже 8.7

Запуск тестов проекта

```bash
make test
```
